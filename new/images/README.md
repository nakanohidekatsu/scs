# プレースホルダー画像について

このモックアップでは、商工中金情報システムの実写真は一切使用していません。すべての写真枠は `css/style.css` の `.ph` クラスによるCSSグラデーションブロックで代替しています。

## 代替している画像とその想定内容

| 画面上のラベル | 実際に入る想定の写真 |
|---|---|
| HERO VISUAL（トップ） | オフィス外観・プロジェクトの様子など、力強く静止感のある1カット（カルーセルではなく単一ビジュアル想定） |
| 代表ポートレート（vision.html） | 社長のポートレート |
| PORTRAIT（people.html ×4） | 各社員のポートレート |

外部サイト（shochu-comp.co.jp）からの画像ダウンロード・ホットリンクは行っていません。実データに差し替える際は、正式に許諾を得た写真素材を用意してください。撮影方針としては、現行サイトの柔らかい雰囲気の写真ではなく、より引き締まった・構図の効いた撮影（オフィスの俯瞰、集中して作業する手元、対話するミーティングの様子など）を推奨します。
