# プレースホルダー画像について

このモックアップでは、商工中金情報システムの実写真は一切使用していません。すべての写真枠は `css/style.css` の `.ph` クラスによるCSSグラデーションブロックで代替しています。

## 代替している画像とその想定内容

| 画面上のラベル | 実際に入る想定の写真 |
|---|---|
| メインビジュアル（カルーセル5枚） | オフィス外観・社員の働く様子など5カット |
| メッセージ写真 | 社長・先輩社員のポートレート |
| 会社案内写真 | オフィスや会議の様子 |
| 福利厚生写真 / 社食写真 | 社員食堂の様子 |
| オフィス写真 | 実際の執務室 |
| 社員写真（インタビュー） | 各社員のポートレート4名分 |

外部サイト（shochu-comp.co.jp）からの画像ダウンロード・ホットリンクは行っていません。実データに差し替える際は、正式に許諾を得た写真素材を用意してください。
